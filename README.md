# Social-Media-Students-Analysis
Exploration and visualization of students' social media usage using **R** and the **tidyverse** package.
The goal of this project is to examine students' behavior regarding social media, compare usage between Single, Complicated, and In Relationship students, and investigate relationships with addiction levels, sleep quality, mental health, and age.

---

## Project Structure
* **data/Students Social Media Addiction.csv**
* **R/socialmedia_eng.R**
* **output/** → Folder for saving generated graphs

---

## Dataset Overview

This dataset contains information about students’ social media usage and its effects on well-being.

---

## Analysis Overview

The analysis includes the following steps:

1. Descriptive Statistics

2. Comparison of Relationship Status Groups

3. Relationship between Avg_Daily_Usage_Hours and Mental_Health_Score

4. Relationship between Avg_Daily_Usage_Hours and Sleep_Hours_Per_Night

5. Relationship between Addicted_Score and Conflicts_Over_Social_Media

6. Relationship between Mental_Health_Score and Sleep_Hours_Per_Night

7. Relationship between Age and Conflicts_Over_Social_Media
